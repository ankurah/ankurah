# Glossary

The maintained glossary lives in the Ankurah book:
https://ankurah.org/glossary.html

For the concurrency terminology specifically (events, clocks, heads, meets,
merge semantics), see https://ankurah.org/concurrency/index.html and the
contributor internals chapters at https://ankurah.org/internals/event-dag.html.

The PUBLISHED book tracks the latest release. For work merged but not yet
released, the same internals chapters live in this repository under
`docs/internals/` and are the tip-of-tree source of truth; read them there
when the published pages lag (for example, event generations and the
comparison accelerations from #266 appear in `docs/internals/event-dag.md`
before the release that publishes them).
